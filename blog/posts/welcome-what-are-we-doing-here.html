Welcome to "What are we doing here?"

Tech & AI January 7, 2025 3 min read

Welcome to my new blog, "What are we doing here?" - a question I find myself asking daily as I navigate the rapidly evolving landscape of technology, finance, and personal wellness.

This blog is born from a simple observation: we're living through an unprecedented convergence of accessible AI tools, cloud technologies, and data analytics platforms that were once reserved for enterprise use. The question isn't whether we should be using these tools for personal projects - it's how we can leverage them meaningfully.

What You'll Find Here

Tech & AI Focus: I'll be documenting my journey with tools like Claude Code, Databricks, and modern cloud platforms. Expect deep dives into how AI-assisted development is changing the way we build and think about software.

Finance Through Data: Personal finance meets data science. I'll explore how to use enterprise-grade analytics tools for personal financial insights, budgeting with SQL, and building dashboards that actually help make better money decisions.

Fitness Tracking Revolution: Moving beyond simple step counters to building comprehensive health analytics using cloud databases, automated tracking, and data visualization tools that provide actionable insights.

The Databricks Connection

Why Databricks? Because it represents something fascinating - a platform designed for massive enterprise data that's becoming accessible for personal use. When you can spin up a lakehouse architecture to analyze your workout data alongside your financial transactions, you start to see patterns that simple apps miss.

I'll be exploring practical applications like:

  • Connecting Neon PostgreSQL fitness data to Databricks for advanced analytics
  • Building personal dashboards that combine health, financial, and productivity metrics
  • Using AI-powered insights to make better life decisions

Built with Claude Code

This entire blog and the underlying portfolio site were built using Claude Code - Anthropic's AI-powered development assistant. Every post will include insights into how AI-assisted development is changing not just what we build, but how we think about building.

So, what are we doing here? We're experimenting, learning, and pushing the boundaries of what's possible when powerful tools meet personal curiosity. Join me as we explore this intersection of technology, data, and daily life.

Next up: Setting up Neon PostgreSQL for fitness tracking and preparing the data pipeline for Databricks integration.
